== Description ==

Have you ever accidentally published a post before it was ready? Yeah. It sucks. This plugin takes care of that.

When you hit the "Publish" button after writing a post, a small dialog box appears asking you if you're sure you want to publish the post. You can either click OK or CANCEL. And that's it! Super simple and no more acidentally published posts.

[Click here for screenshots and video](http://www.bloggingbookshelf.com/wordpress/publish-confirmation-plugin/ "Click here for screenshots and video") 


== Installation ==

1. Download plugin.
2. From the WordPress administration dashboard, go to Plugins > Add New > Upload. 
3. Select file, click "Install Now."
4. Activate plugin.
5. Plugin installation is complete after you've clicked "Install Now." You don't need to do anything else.

OR

1. Download plugin.
2. Upload zipped plugin file to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. See step 5 above.
